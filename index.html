Welcome to MAFRA ( Managed File Replication Assistant )
What is MAFRA?

MAFRA is a tool which main purpose is to assist NET TSEs with Managed File Replication issues. MAFRA helps with two types of issues: MFR failure, and lack of performance (cloning is slow) type of issues.
