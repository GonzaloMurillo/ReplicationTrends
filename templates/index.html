Welcome to Replication Trends
What is Replication Trends?

Replication Trends is a tool that processes the info of autosupport files, and displays the replication metrics for every context found.

Replication Trends plots a fancy graph for every context, and can estimate when a context will be in sync.

Please start by providing the path to the autosupport files within the evidence server, something like: /auto/cores/cxx/ddr/var/support
