Welcome to MAFRA ( Managed File Replication Assistant )
What is MAFRA?

MAFRA is a tool which main purpose is to assist NET TSEs with Managed File Replication issues.

MAFRA helps with two types of issues: failure to clone, and lack of performance (cloning is slow) issues.

MFR is used by Backup Software like Networker (where it is called cloning) and NetBackp (where it is called SLP)
