What is Managed File Replication Assistant?

Managed File Replication Assistant is a tool which main purpose is to assist NET TSEs with Managed File Replication issues. Managed File Replication Assistant can help you with two types of issues: MFR failure, and lack of performance (cloning is slow) issues.
